# AgeraBus
An event bus implements with Agera
### Usage:
Add dependency using maven:
```
<dependency>
  <groupId>com.cmos</groupId>
  <artifactId>agerabus</artifactId>
  <version>2.0.0</version>
  <type>pom</type>
</dependency>
```
#### Or
add dependency using gradle:
```
compile 'com.cmos:agerabus:2.0.0'
```
